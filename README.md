# RetailIQ: End-to-End Retail Sales Analytics Pipeline

## Problem Statement

This project aims to perform a comprehensive analysis of sales data from a fictional retail superstore to generate actionable insights that support strategic business decision-making. The primary challenges addressed include:

- Preprocessing and cleaning raw, unstructured data  
- Transforming data into an analysis-ready format  
- Conducting exploratory data analysis (EDA) to uncover patterns, trends, and key performance indicators (KPIs)  
- Presenting insights through intuitive and interactive visualizations  

---

## Solution Overview

An end-to-end data analytics pipeline has been implemented by integrating Python, SQL, and Power BI to ensure efficient data processing, analysis, and visualization.

### Data Preprocessing (Python)
Python is utilized for data cleaning, validation, and transformation. The original Excel dataset is processed to handle inconsistencies, missing values, and formatting issues, and subsequently converted into a structured CSV format.

### Data Storage & Querying (SQL)
The cleaned dataset is imported into a relational SQL database. Structured Query Language (SQL) is used to perform data manipulation and execute analytical queries for extracting meaningful insights.

### Data Visualization (Power BI)
Power BI is employed to design interactive dashboards and reports. These visualizations enable efficient exploration of business metrics and facilitate data-driven decision-making.

---

## Repository Structure

The repository is organized as follows:

- **Python Scripts**  
  Scripts for data cleaning, preprocessing, and transformation of the raw Excel dataset.

- **SQL Files**  
  - SQL dump file for database import  
  - Analytical queries for exploratory data analysis and insight generation  

- **Power BI Dashboard (.pbix)**  
  An interactive reporting file comprising three key analytical views:
  - Sales Performance Analysis  
  - Customer Segmentation & Behavior Analysis  
  - Product-Level Performance Analysis
